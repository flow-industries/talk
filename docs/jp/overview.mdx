## はじめに

Flow Talk は、世界で最も安全で分散化されたソーシャルネットワークです。
あなたは自分のアイデンティティ、つながり、コンテンツを完全にコントロールでき、どこへでも持ち運べます。

Flow Talk は[オープンソースソフトウェア](https://github.com/flow-industries/talk)です。
オープンで透明、そして本当にユーザーに属するソーシャルの未来を、ぜひ一緒に作っていきましょう。

## なぜ今？

これまでの分散型ソーシャルは、「十分な」分散化を目指し、速度・コスト・セキュリティのどれかを妥協せざるを得ませんでした。

いまは、妥協なしで「完全な」分散化が可能です。イーサリアムのスケーリングによって、ブロックチェーンへ直接書き込み、イーサリアムをデータ可用性レイヤーとして使えるようになりました。

Flow Talk は、任意の EVM 互換チェーンにデプロイできるコントラクト群の集合に過ぎません。
物事はシンプルであるべき——まさにそのとおりの設計です。

これにより、Flow Talk は初日から世界で最も安全で、恒久的で、完全に分散化されたソーシャルネットワークとなります。

今後、Ethereum L1 と各種 L2 のスケールが進むにつれて、セキュリティや分散性を損なうことなく、Flow Talk での投稿コストはより低くなっていきます。
